# How to Get Your Gemini API Key

## Step-by-Step Guide

### 1. Visit Google AI Studio

Go to: **https://aistudio.google.com/apikey**

### 2. Sign In

- Click "Sign in" if not already signed in
- Use your Google account
- Any Google account works (Gmail, Workspace, etc.)

### 3. Create API Key

- Click **"Create API Key"** button
- Choose "Create API key in new project" (recommended)
- Or select an existing Google Cloud project

### 4. Copy Your Key

- Your API key will be displayed
- It starts with `AIza`
- Click the copy icon to copy it
- **Important:** Save it somewhere safe!

### 5. Add to ResumeForge

1. Login to ResumeForge
2. Click **Settings** in the navigation
3. Find "Gemini API Key" section
4. Paste your API key
5. Click **Save**
6. Wait for validation
7. See "✅ API key saved successfully"

### 6. Start Using

You can now:
- Analyze GitHub repositories
- Score library items against job descriptions
- Generate tailored resumes

## Troubleshooting

### "Invalid API key format"
**Problem:** Key doesn't start with `AIza`
**Solution:** 
- Copy the entire key
- Make sure no spaces before/after
- Key should be ~39 characters

### "API key validation failed"
**Problem:** Key is not valid or not enabled
**Solution:**
- Go back to Google AI Studio
- Check key is enabled
- Try creating a new key
- Make sure you copied the full key

### "Rate limit exceeded"
**Problem:** You've hit your daily/minute limit
**Solution:**
- Wait a few minutes (for per-minute limit)
- Wait until tomorrow (for daily limit)
- Or upgrade to paid tier (if needed)

### Can't access Google AI Studio
**Problem:** Site not loading or access denied
**Solution:**
- Check your internet connection
- Try different browser
- Make sure you're signed into Google
- Clear browser cache

## FAQ

**Q: Is it really free?**
A: Yes! Google provides a generous free tier for Gemini API.

**Q: Do I need a credit card?**
A: No, the free tier doesn't require payment info.

**Q: What if I hit the limits?**
A: You can wait for the limit to reset, or upgrade to paid tier.

**Q: Can I use the same key on multiple devices?**
A: Yes, but it's your personal key. Don't share it publicly.

**Q: How do I revoke a key?**
A: Go to Google AI Studio → API Keys → Delete the key.

**Q: Can I create multiple keys?**
A: Yes, you can create multiple keys for different projects.

**Q: What if I lose my key?**
A: Just create a new one and update it in ResumeForge Settings.

**Q: Is my data sent to Google?**
A: Yes, when using AI features. That's how the AI works. Google's privacy policy applies.

**Q: Can I use a different AI provider?**
A: Currently only Gemini is supported. Other providers may be added later.
